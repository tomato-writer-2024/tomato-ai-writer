# 域名备案说明

## 当前域名

**tomato-ai-writer.vercel.app**

## 备案状态

### ✅ 不需要备案

**原因说明**：

1. **Vercel提供的域名**
   - `tomato-ai-writer.vercel.app` 是Vercel提供的子域名
   - Vercel的托管服务器位于海外（美国、欧洲、亚太等地区）
   - 使用Vercel的域名**不需要**进行ICP备案

2. **ICP备案适用范围**
   - 中国大陆境内服务器托管的网站需要ICP备案
   - 使用中国大陆云服务（如阿里云、腾讯云、华为云）需要备案
   - 使用海外服务器（如Vercel、Netlify、AWS）**不需要**备案

3. **当前部署架构**
   ```
   用户 → 访问 tomato-ai-writer.vercel.app
         ↓
   Vercel全球CDN节点（香港、新加坡、美国等）
         ↓
   Vercel边缘服务器（海外）
   ```

## 如何优化中国大陆用户访问

### 方案1：使用CDN加速（当前方案）
- ✅ Vercel已经内置全球CDN
- ✅ 香港节点、新加坡节点较近
- ⚠️ 仍需经过国际出口

**优点**：
- 免费
- 自动配置
- 全球统一域名

**缺点**：
- 中国大陆访问速度较慢
- 部分网络可能受限

### 方案2：使用自定义域名（不需要备案）
**条件**：
- 使用Vercel提供的 `.vercel.app` 域名
- 或使用未在中国大陆注册的域名（如 `.io`, `.ai`, `.dev`）

**优点**：
- 更专业的域名
- 品牌一致性

**缺点**：
- 需要购买域名（$10-50/年）
- 中国大陆访问速度问题仍存在

### 方案3：使用国内服务器 + ICP备案（需要备案）
**条件**：
- 购买中国大陆域名（如 `.cn`, `.com.cn`）
- 使用中国大陆云服务器（阿里云、腾讯云等）
- 完成ICP备案（需要20-30个工作日）

**优点**：
- ✅ 中国大陆访问速度快
- ✅ 访问稳定，不易被屏蔽
- ✅ 符合中国法律法规

**缺点**：
- ❌ 需要ICP备案（流程繁琐）
- ❌ 需要购买国内服务器（成本高）
- ❌ 需要单独配置国内服务器
- ❌ 维护复杂度增加

### 方案4：双域名方案（推荐）
- **主域名**：`tomato-ai-writer.vercel.app`（海外用户）
- **国内域名**：`tomato-ai-writer.cn`（中国大陆用户，需要备案）

**优点**：
- 海外用户使用主域名（速度快）
- 国内用户使用国内域名（速度快、稳定）
- 按需访问，自动优化

**缺点**：
- 需要维护两套服务
- 需要ICP备案
- 成本增加

## 备案流程（如需要）

### ICP备案所需材料
1. **主体资格证明**
   - 个人：身份证正反面
   - 企业：营业执照、组织机构代码证

2. **域名证明**
   - 域名证书
   - 域名注册人信息

3. **服务器证明**
   - 云服务器购买凭证
   - 服务器IP地址

4. **其他材料**
   - 网站备案申请表
   - 负责人身份证
   - 网站负责人照片

### ICP备案流程
1. **准备材料**（1-3天）
   - 收集上述所有材料
   - 填写备案申请表

2. **提交审核**（1天）
   - 登录云服务商备案系统
   - 提交备案申请

3. **服务商初审**（1-3天）
   - 云服务商审核材料完整性
   - 核实服务器信息

4. **管局审核**（15-20天）
   - 省通信管理局审核
   - 可能需要核验

5. **备案完成**（1天）
   - 收到备案号
   - 域名备案成功

**总计时间**：20-30个工作日

- **云服务器**：¥100-500/月（根据配置）
- **域名**：¥50-100/年
- **备案**：免费（部分服务商收费¥50-200）

## 常见问题

### Q1: vercel.app域名会被封吗？
**A**: 一般不会被永久封禁。但偶尔可能因为：
- GFW临时限制
- 运营商限速
- 网络波动

**解决方法**：
- 等待自动恢复（通常几小时到几天）
- 刷新DNS缓存
- 使用VPN/代理访问

### Q2: 是否需要备案才能运营？
**A**: 不一定。根据情况：

**不需要备案**：
- 主要面向海外用户
- 使用Vercel等海外服务
- 使用`.vercel.app`等域名

**需要备案**：
- 主要面向中国大陆用户
- 使用中国大陆服务器
- 使用`.cn`、`.com.cn`等国内域名
- 涉及商业运营

### Q3: 如果国内用户无法访问怎么办？
**A**: 提供多种解决方案：

2. **中期**

3. **长期**

### Q4: 如何提高国内访问速度？
