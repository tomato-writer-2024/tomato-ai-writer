## 🎯 5分钟快速上手

### 第一步：访问应用
打开浏览器，访问：
```
https://tomato-ai-writer.vercel.app
```

### 第二步：注册账号
1. 点击右上角"免费注册"
2. 填写注册信息：
   - 用户名：`yourname`
   - 邮箱：`your@email.com`
   - 密码：`YourPassword123`
3. 点击"注册"
4. 自动登录，进入工作空间

### 第三步：创建第一个角色
1. 在工作空间，点击"角色生成器"
2. 填写表单：

3. 点击"生成角色"

### 第四步：生成大纲
1. 返回工作空间，点击"大纲生成器"
2. 填写表单：
   - 小说标题：`《混沌战神》`
   - 题材：`玄幻`

   - 章节数量：`300章`

3. 点击"生成大纲"
4. 查看AI生成的完整大纲和章节规划

1. 进入"我的作品"
2. 点击"创建新作品"
3. 填写作品信息
4. 开始撰写第一章

## 📖 功能使用指南

### 一、角色设定

#### 1. 角色生成器
**使用场景**：需要创建新角色时

**操作步骤**：
1. 进入"角色生成器"
2. 填写基本信息：
   - 角色名称（必填）
   - 角色类型（主角/配角/反派/辅助）
   - 题材（玄幻/武侠/仙侠/都市/历史/军事/科幻/灵异）
   - 故事背景（可选）
3. 点击"生成角色"
4. 等待2-3秒，查看生成的角色设定
5. 如不满意，可重新生成或手动编辑

**生成结果包含**：
- 基本设定（姓名、身份、外貌）
- 性格特征
- 背景故事
- 动机目标
- 能力设定
- 人际关系
- 成长轨迹

**提示**：
- 故事背景越详细，生成的角色越立体
- 可以多次生成，选择最喜欢的版本
- 生成的角色可以保存到角色库

#### 2. 人物关系图
**使用场景**：理清复杂的人物关系

**操作步骤**：
1. 进入"人物关系图"
2. 点击"添加角色"
3. 输入角色名称
4. 拖拽角色节点到合适位置
5. 点击两个角色，建立关系
6. 选择关系类型（朋友/敌人/师徒/亲属/恋人）
7. 添加关系描述
8. 保存关系图

**提示**：
- 支持拖拽调整位置
- 支持缩放查看
- 可导出为图片

### 二、情节设计

#### 1. 大纲生成器
**使用场景**：开始新小说时

**操作步骤**：
1. 进入"大纲生成器"
2. 填写基本信息：
   - 小说标题（必填）
   - 题材（必填）
   - 目标字数（必填）
   - 章节数量（必填）
   - 核心冲突（必填）
   - 故事背景（可选）
3. 点击"生成大纲"
4. 等待3-5秒，查看生成的大纲
5. 查看章节规划，可调整章节顺序
6. 保存大纲

**生成结果包含**：
- 故事大纲
- 章节规划（每章标题、字数、剧情点）
- 主线剧情
- 副线设定
- 高潮节点
- 结局设定

**提示**：
- 核心冲突越清晰，大纲越精彩
- 可以选择预设模板
- 大纲生成后可手动编辑

#### 2. 情节反转
**使用场景**：需要制造剧情反转时

**操作步骤**：
1. 进入"情节反转"
2. 填写当前情节
3. 选择反转类型：
   - 身份反转
   - 能力反转
   - 情感反转
   - 时间反转
   - 空间反转
4. 选择反转强度（1-5星）
5. 输入目标效果
6. 点击"生成反转"
7. 查看多个反转方案

**提示**：
- 反转强度越高，剧情越震撼
- 可以生成多个方案，选择最合适的
- 反转会影响后续剧情，需谨慎选择

#### 3. 世界观构建
**使用场景**：需要搭建完整世界设定

**操作步骤**：
1. 进入"世界观构建"
2. 填写世界基本信息：
   - 世界名称
   - 世界类型（奇幻/科幻/都市/历史/架空）
   - 世界规模
   - 世界背景
3. 点击"生成世界观"
4. 查看生成的世界设定
5. 逐项完善：
   - 地理环境
   - 势力分布
   - 修炼/科技体系
   - 货币系统
   - 语言文化
   - 历史事件

**提示**：
- 世界观设定会影响后续创作
- 可以参考热门作品的世界观
- 保持世界观一致性

### 三、智能写作

#### 1. 智能续写
**使用场景**：写不下去时

**操作步骤**：
1. 进入"智能续写"
2. 粘贴当前段落
3. 选择续写长度（500字/1000字/2000字）
4. 选择续写风格（严肃/轻松/热血/悬疑）
5. 设定剧情方向（可选）
6. 点击"生成续写"
7. 查看多个续写方案
8. 选择最喜欢的方案

**提示**：
- 当前段落越长，续写越准确
- 剧情方向可以引导AI
- 可以多次生成，选择最佳方案

#### 2. 黄金开头
**使用场景**：需要吸引读者时

**操作步骤**：
1. 进入"黄金开头"
2. 填写基本信息：
   - 小说题材
   - 主角类型
   - 开头类型（悬念/冲突/奇遇/穿越/重生）
   - 核心卖点
3. 点击"生成开头"
4. 查看多个开头方案
5. 查看每个开头的吸引力分析
6. 选择最合适的开头

**提示**：
- 黄金开头要在3秒内吸引读者
- 可以参考爆款作品的开头
- 开头要设置悬念或冲突

#### 3. 结局生成
**使用场景**：需要完美收官时

**操作步骤**：
1. 进入"结局生成"
2. 输入故事大纲摘要
3. 设定主角最终状态
4. 选择结局类型：
   - 大团圆
   - 悲剧
   - 开放式
   - 悬念
5. 输入核心伏笔（可选）
6. 点击"生成结局"
7. 查看多个结局方案
8. 查看伏笔回收列表

**提示**：
- 结局要呼应开头
- 要回收所有重要伏笔
- 可以设置悬念，为续作留白

#### 4. 风格模拟
**使用场景**：想要模仿某位作家的风格

**操作步骤**：
1. 进入"风格模拟"
2. 选择目标作家
   - 热门作家列表
   - 自定义风格
3. 粘贴原文
4. 点击"转换风格"
5. 查看转换结果
6. 对比原文和转换后的文本

**提示**：
- 可以多次调整风格参数
- 转换后的文本可手动编辑
- 风格模拟仅供参考

## 💡 使用技巧

### 1. 提高AI生成质量
- 输入的信息越详细，生成效果越好
- 参考爆款作品，但不要抄袭
- 可以多次生成，选择最好的版本
- 生成的内容可以手动修改

### 2. 提升创作效率
- 先生成大纲，再写章节
- 使用角色生成器，快速创建角色
- 利用智能续写，突破卡文
- 使用素材管理，收集灵感

### 3. 提升作品质量
- 使用编辑审稿，检查章节质量
- 使用爽感引擎，优化爽点设置
- 使用爆款拆解，学习成功经验
- 关注读者反馈，持续改进

### 4. 避免常见问题
- 不要过于依赖AI，要保持自己的创意
- AI生成的内容要检查，避免不合理的地方
- 保持世界观和人物设定的一致性
- 定期保存作品
